# Trading App - Backend

This is the backend for the Trading App, built with Node.js, Express, and MongoDB.

## Installation

1. Clone the repository.
2. Navigate to the `app_v2/backend` directory.
3. Create a `.env` file based on `.env.example` and fill in your details.
4. Run `npm install` to install dependencies.

## Running the app

```bash
# development
npm run dev

# production
npm start
```

## API Documentation

API documentation is available at `/api-docs` when the server is running.

## Deployment

This app is configured for deployment on Heroku.
