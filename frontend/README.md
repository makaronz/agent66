# Trading App - Frontend

This is the frontend for the Trading App, built with React, Redux, and Tailwind CSS.

## Installation

1. Clone the repository.
2. Navigate to the `app_v2/frontend` directory.
3. Create a `.env` file based on `.env.example` and set the `REACT_APP_API_URL`.
4. Run `npm install` to install dependencies.

## Running the app

```bash
# development
npm start
```

This runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Deployment

This app is configured for deployment on Vercel.
